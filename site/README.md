# Análisis de compra de vivienda · Bogotá

Sitio estático listo para GitHub Pages.

## Cómo publicarlo (5 minutos)
1. Crea un repositorio **público** en GitHub (ej. `analisis-vivienda`).
2. Sube **todo el contenido de esta carpeta** (el `index.html` y la carpeta `img/`).
   - Vía web: botón "Add file" → "Upload files" → arrastra todo → Commit.
3. En el repo: **Settings → Pages**.
4. En "Source" elige la rama `main` y carpeta `/ (root)` → **Save**.
5. Espera ~1 minuto. Tu sitio queda en:
   `https://TU_USUARIO.github.io/analisis-vivienda/`

Listo. Cualquier cambio que subas al repo se publica solo.
